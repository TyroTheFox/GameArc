A header file that contains global external objects that can then be used by any class in the project. These are the \mbox{\hyperlink{class_model}{Model}} Handler, The \mbox{\hyperlink{class_input}{Input}} Handler and the Key \mbox{\hyperlink{class_event}{Event}}. These are external objects that are accessable by all classes but must be initialised before use.
