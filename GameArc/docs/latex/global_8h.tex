A header file that contains global external objects for the \hyperlink{class_model}{Model} Handler, the Input Handler and the \hyperlink{class_key}{Key} \hyperlink{class_event}{Event} objects. 

A header file that contains global external objects that can then be used by any class in the project. These are the \hyperlink{class_model}{Model} Handler, The Input Handler and the \hyperlink{class_key}{Key} \hyperlink{class_event}{Event}. These are external objects that are accessable by all classes but must be initialised before use.
